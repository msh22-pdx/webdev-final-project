# Intro to Web Development Final Project: Personal Portfolio

**Deployed Site:** [https://msh22-pdx.github.io/webdev-final-project/](https://msh22-pdx.github.io/webdev-final-project/)

## Project Description

This is a professional personal portfolio website created for CS 463/563: Intro to Web Development. It serves as a single-page application highlighting my background, previous work, and selected projects.

## How to Run the Code

1. Clone this repository to your local machine.
2. Open the `index.html` file in your preferred web browser.
3. No build steps or local servers are required to view the basic site.
